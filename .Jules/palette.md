## 2024-05-22 - [Copy to Clipboard]
**Learning:** Developers frequently need to copy-paste code snippets from documentation. Adding a dedicated "Copy" button reduces friction and improves the developer experience significantly.
**Action:** When designing documentation pages, always include a "Copy" button for code blocks.

## 2025-05-25 - [Contextual Actions]
**Learning:** Text data like API URLs are often manually selected and copied by users. Extending the "Copy" functionality to these non-code-block data elements (like endpoints) saves user effort and prevents selection errors.
**Action:** Identify static text data (URLs, keys, IDs) that users frequently copy and attach copy actions to them.
