## 2024-05-22 - [Copy to Clipboard]
**Learning:** Developers frequently need to copy-paste code snippets from documentation. Adding a dedicated "Copy" button reduces friction and improves the developer experience significantly.
**Action:** When designing documentation pages, always include a "Copy" button for code blocks.

## 2024-10-24 - [API Endpoint UX]
**Learning:** Users often need to copy API URLs directly. Providing a copy button next to the URL is a high-value micro-interaction that saves frustration compared to selecting text manually.
**Action:** Ensure all display-only URLs in documentation or dashboards have an associated copy button.
