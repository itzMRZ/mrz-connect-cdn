## 2024-05-22 - [Copy to Clipboard]
**Learning:** Developers frequently need to copy-paste code snippets from documentation. Adding a dedicated "Copy" button reduces friction and improves the developer experience significantly.
**Action:** When designing documentation pages, always include a "Copy" button for code blocks.

## 2026-01-22 - [Unified Copy UX]
**Learning:** Users often need to copy static text (like API URLs) just as much as code snippets. Inconsistent copy behavior (button on code, manual select on text) causes friction.
**Action:** Implement a generic "Copy" button utility that can be attached to any element (code blocks, endpoints, config values) to provide a consistent experience across different data types.
