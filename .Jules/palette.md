## 2024-05-22 - [Copy to Clipboard]
**Learning:** Developers frequently need to copy-paste code snippets from documentation. Adding a dedicated "Copy" button reduces friction and improves the developer experience significantly.
**Action:** When designing documentation pages, always include a "Copy" button for code blocks.

## 2024-05-23 - [Reuse Copy Patterns]
**Learning:** Developers need to copy API endpoints as much as code snippets. Reusing the 'copy to clipboard' pattern for non-code text elements (like URLs) is a high-value, low-effort UX win.
**Action:** When adding copy functionality, abstract it to accept any container/selector pair, not just pre/code blocks.
