## 2024-05-22 - [Copy to Clipboard]
**Learning:** Developers frequently need to copy-paste code snippets from documentation. Adding a dedicated "Copy" button reduces friction and improves the developer experience significantly.
**Action:** When designing documentation pages, always include a "Copy" button for code blocks.

## 2024-05-22 - [Actionable API Endpoints]
**Learning:** Static lists of API endpoints are less useful if they can't be easily copied. Adding "Copy URL" buttons to endpoint cards transforms them from passive information to actionable tools.
**Action:** Any UI element displaying a URL or ID that a user likely needs to use elsewhere should have a one-click copy action.
